# Image-Feed-Browser

An Android application showing photos retrieved from Flickr upon user search requests.

Search for high quality photos from Flickr.

Showing thumbnails of the pictures and magnify the selected picture.

Created using:

              - Flickr API
              
              - Java
              
              - Android Studio
